# Sentor_G | Platinum Kaizo Nuzlocke Adjustments

## Being so heavily involved with the IronMON community and my personal runs, I have never attempted a Nuzlocke. I did something similar on my PokeMMO alt account (the [Senlocke](https://github.com/SentorG/PokeMMOSenlocke/blob/main/README.md)) where I beat all regions with only water-type Pokémon. To make it work, I made adjustments to the [general Nuzlocke rules](https://nuzlockeuniversity.ca/nuzlocke-rules/) to allow me to do what I wanted to without bankrupting my main account.<br><br>The [Platinum Kaizo](https://platinumkaizo.com/) ROM hack was released on March 15th 2026 and I am eager to try it. Below are listed the changes I want to employ to embark on a journey to explore this ROM hack without the pressure of being "[Hardcore](https://nuzlockeuniversity.ca/nuzlocke-variants/hardcore-nuzlocke-variant/)" or a "[Hall of Fame](https://platinumkaizo.com/hall-of-fame-rules/)" attempt - I might come back to those one day if I enjoy the run.

## Rules:
| **Rule** | **Details** |
|:-:|:-|
| **Fresh Run** | When starting a new run, a new Trainer ID must be generated. |
| **Hard Mode** | Easy Mode is banned and must not be enabled at any point during the run. |
| **Set Mode** | Set Mode must be enabled for the entire run. |
| **Wheel's Choice** | Instead of taking the first encounter on a Route, a wheel spin will decide **WHICH NUMBER ENCOUNTER** will be chosen. Encounter rule details can be found [here](https://platinumkaizo.com/hall-of-fame-rules/#encounter-rules). |
| **Supercharged** | May use PKHex to manually apply the maximum amount of EVs desired and can be changed at any point.<br>**MUST NOT** change IVs or Nature. |
| **Blessed Economy** | May use PKHex to add any item & quantity to the Bag. |
| **5 Items Per Battle** | Only five items from the Bag, including Ball-stall turns, are permitted for each battle. |
| **Friends Forever** | If a Pokémon faints, they are not considered "dead". |
| **Hardcore Level Cap** | Must adhere to the Hall of Fame run [level cap guide](https://platinumkaizo.com/hall-of-fame-rules/#level-caps). |
